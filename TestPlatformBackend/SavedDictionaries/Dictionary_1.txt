HTML - стандартна мова розмітки для створення веб-сторінок.

XML - розширювана мова розмітки, яка стосується спеціалізованих мов розмітки.

MathML - мова розмітки для представлення математичних символів і формул в документах WWW.

XHTML - розширювана мова гіпертекстової розмітки, яка є більш строгою версією HTML.

<!DOCTYPE> - команда, яка вказує веб-браузеру версію HTML або XHTML, в якій написана веб-сторінка.

Елемент - будівельний блок веб-сторінки, який складається з відкриваючого тегу, вмісту і закриваючого тегу.

Тег - слово або символ, що використовується в HTML для позначення початку і кінця елементів.

Атрибути тегів - додаткові характеристики або параметри, які можуть бути додані до тега для надання додаткової інформації або функціональності.

Теги <b>, <i>, <h1> до <h6> - різні теги для форматування тексту на веб-сторінці, від жирного та курсиву до заголовків різного рівня.

Контейнер <div> - блоковий контейнер, який використовується для групування елементів для стилізації або іншої логічної групи.

Таблиці - структуровані теги, такі як <table>, <tr>, і <td>, для створення та управління табличними даними на веб-сторінці.

<!DOCTYPE html> - сучасний стандарт декларації типу документу для HTML5.
