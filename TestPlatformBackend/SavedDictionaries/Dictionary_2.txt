Тег - будівельний блок у HTML, вказує на тип елемента.
Елемент - конструкція на веб-сторінці, складається з тегів і містить контент.
Браузер - програма для перегляду веб-сторінок, відображає елементи згідно з тегами.
Пошукові системи - сервіси для пошуку інформації в інтернеті, індексують структуру та зміст веб-сторінок.
Заголовки - спеціальні елементи в HTML (наприклад, <h1>...<h6>), використовуються для ієрархічного представлення інформації на сторінці.
Таблиці - елементи для організації даних у табличному вигляді на веб-сторінці.
Списки - елементи для представлення упорядкованих або неупорядкованих даних.
Зображення - елементи для вставки графічних зображень на веб-сторінку.
Атрибути - спосіб надання додаткових інструкцій або налаштувань для HTML-елементів.
Контейнер - об'єкт в HTML, який використовується для структурування і групування іншого контенту на веб-сторінці для стилізації або управління багатьма елементами як єдиним об'єктом.
XHTML - версія HTML, що використовує стандарти XML, більш строга ніж звичайний HTML.
Цвет - властивість в HTML та CSS для задання кольору елементів на сторінці; може встановлюватися за допомогою назви кольору, шістнадцяткового коду або RGB-значень.
DOCTYPE - декларація у документі HTML, що показує версію і тип HTML або XHTML документа, який ви використовуєте.
HTML - HyperText Markup Language, мова розмітки для створення веб-сторінок.
CSS - Cascading Style Sheets, мова стилів, яка використовується для задання візуального оформлення веб-сторінки.
