SVG - Відкритий формат векторної графіки, який розроблено на основі XML для опису двовимірних графічних зображень.

Canvas - HTML-елемент, який використовується для малювання графіки на льоту за допомогою скриптів (зазвичай JavaScript).

DOM (Document Object Model) - Об'єктна модель документу, яка дозволяє програмам змінювати структуру, стиль та вміст документів.

HTML Video - Елемент, що визначає стандартний спосіб вбудовування відео на веб-сторінки.

HTML Audio - Елемент, який визначає стандартний спосіб вбудовування звуку на веб-сторінки.

Градієнт - Спосіб забарвлення, що використовується для поступового переходу від одного кольору до іншого.

Геометричні примітиви - Основні фігури, такі як точки, лінії, криві та багатокутники, які використовуються для створення складніших форм.

Атрибути SVG - Властивості, які визначають характеристики SVG елементів, наприклад, кольори, розміри, форми тощо.

Retained Mode Graphics - Технологія малювання, яка зберігає деталі об'єктів у пам’яті для зручності манипуляцій з ними.

Linear Gradient - Лінійний градієнт; використовується для створення плавного переходу від одного кольору до іншого уздовж прямої лінії.

Radial Gradient - Радіальний градієнт; використовується для створення кольорового переходу з центральної точки наружу.

Web-технології - Набір стандартних способів використання мови розмітки, скриптів, стилів і інших технологій для створення і управління веб-сторінками.

Track Element - Елемент в HTML, призначений для використання текстових доріжок (субтитрів, розділів тощо) у медіа елементах, як-то <audio> або <video>.

Gradient Stops - Колірні точки в градієнті, що визначають колір і його положення уздовж градієнта.
