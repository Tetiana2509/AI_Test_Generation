Запитання 1:
Які комбінатори визначені в CSS для стилізації елементів на основі їхніх відносин у дереві DOM?
Рівень складності: середній
а) Селектор нащадків (пробіл)
б) Селектор дочірній (>)
в) Селектор сусідній споріднений (+)
г) Селектор загальний споріднений (~)
д) Селектор батьківський (^) - неправильно

Правильні відповіді: а, б, в, г

Запитання 2:
Які методи описані у тексті для впливу на стили предків HTML елементів, що не підтримуються напряму в CSS?
Рівень складності: складний
а) За допомогою JavaScript додавання класу або стилів
б) Використання CSS-in-JS для динамічного генерування стилів
в) Вплив через стилізацію нащадків
г) Використовувати :hover та :focus для стилізації предків - неправильно
д) Пряму стилізацію предків через селектори в CSS - неправильно

Правильні відповіді: а, б, в
