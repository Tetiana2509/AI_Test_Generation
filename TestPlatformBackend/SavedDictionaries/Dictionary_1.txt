HTML (Hyper Text Markup Language) - Стандартна мова розмітки для створення веб-сторінок.

XML (eXtensible Markup Language) - Розширювана мова розмітки, призначена для зберігання та передачі даних.

MathML (Mathematical Markup Language) - Мова розмітки, базована на XML для представлення математичних формул у документах WWW.

XHTML (Extensible Hypertext Markup Language) - Розширена мова розмітки гіпертексту, що є більш строгим форматом HTML, визначеним як додаток XML.

DOCTYPE - Декларація типу документа, що вказує версію HTML або XHTML, яку сторінка використовує.

CSS (Cascading Style Sheets) - Мова стилів, яка використовується для опису зовнішнього вигляду HTML документів.

Семантичний елемент - Елемент HTML, що явно вказує на значення частины контенту відповідно до її значущості.

Атрибути - Параметри в тегах HTML, які надають додаткову інформацію про елемент, наприклад, стиль або поведінку.

Тег - Синтаксична конструкція мови розмітки, що огортає контент на сторінці, вказуючи його структуру або форматування.

Елемент - Основна одиниця структури HTML-документу, яка складається з відкриваючого тегу, закриваючого тегу і вмісту між ними.

Посилання - Елемент, що дозволяє користувачам переходити з одного ресурсу на інший за допомогою елемента <a>.

Контейнер - Елемент HTML, який використовується для групування інших елементів для організації контенту чи стилів.

Списки - Елементи для представлення упорядкованих (ol), неупорядкованих (ul) або визначених (dl) списків значень чи термінів.

Таблиці - Використовуються для структурування даних у вигляді сітки, що складається з рядків (tr) та комірок (td або th).

Зображення - Вставка растрових або векторних зображень на сторінку через тег <img>, куди включено атрибути для визначення шляху до зображення, його розмірів та альтернативного тексту.
