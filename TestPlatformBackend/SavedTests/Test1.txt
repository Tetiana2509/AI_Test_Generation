1. Які елементи HTML використовуються для створення таблиць? (Рівень складності: середній)
   - A) <table>, <tr>, <td>
   - B) <table>, <thread>, <tab>
   - C) <list>, <li>, <ul>
   - D) <div>, <span>, <style>
   - E) <header>, <footer>, <main>

   Правильна відповідь: A

2. Що означає атрибут "colspan" у HTML? (Рівень складності: середній)
   - A) Вказує колір стовпця
   - B) Об'єднує декілька рядків у один
   - C) Встановлює ширину колонки
   - D) Об'єднує декілька стовпців у один
   - E) Встановлює кількість колонок у таблиці

   Правильна відповідь: D

3. Чому слід використовувати тег <div> у HTML? (Рівень складності: середній)
   - A) Для вбудовування відео
   - B) Для додавання скриптів
   - C) Для структурування контенту і стилізації
   - D) Тільки для створення гіперпосилань
   - E) Тільки для вставки зображень

   Правильна відповідь: C
